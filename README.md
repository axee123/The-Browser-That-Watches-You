📝 Project Summary

Developed a browser-based psychological horror game titled “The Browser That Watches You”, where the gameplay dynamically reacts to user interaction, simulating the feeling of being observed while playing. The project blends web development with interactive storytelling to create an immersive horror experience using standard web technologies.

The game is built using a full-stack web architecture where frontend elements (HTML, CSS, JavaScript) handle the gameplay interface and user interactions, while PHP and MySQL manage backend logic, user sessions, and game state persistence.

The core concept revolves around a “watching entity” that responds to player behavior such as inactivity, cursor movement, repeated actions, and decision patterns—creating adaptive horror events in real-time.

🛠️ How to Run the Project
Install XAMPP/WAMP Server
Place project folder inside htdocs directory
Start Apache & MySQL from control panel
Import database using phpMyAdmin
Update database connection in config.php

Open browser and run:

http://localhost/watcher-game/
Start the game from homepage
Enable JavaScript for full experience
